## 📖 Описание проекта

**Anime Edit Bot** — это Telegram-бот, написанный на Python, который позволяет создавать стильные видео-эдиты в аниме-стилистике. Бот автоматически синхронизирует нарезку клипов с битами музыки, накладывает эффекты и текст, а на выходе выдаёт готовое вертикальное видео (9:16) для TikTok / Shorts / Reels.

### 🎯 Для кого этот бот?

- Аниме-эдитеры, которые хотят быстро делать эдиты с телефона
- Начинающие монтажёры без опыта работы в сложных программах
- Контент-мейкеры для TikTok и YouTube Shorts

## 📦 Установка на Termux (пошагово)

### Шаг 1: Установка Termux

1. Скачай **Termux** с [F-Droid](https://f-droid.org/packages/com.termux/) (НЕ из Google Play — там устаревшая версия!)
2. Открой Termux и обнови пакеты:

```bash
pkg update && pkg upgrade -y
```

### Шаг 2: Установка системных зависимостей

```bash
# Python и основные инструменты
pkg install python python-pip git ffmpeg -y

# Зависимости для Pillow (работа с изображениями)
pkg install libjpeg-turbo libpng freetype -y

# Зависимости для aubio (бит-детекция)
pkg install libsndfile libfftw -y

# Опционально: nano для редактирования файлов
pkg install nano -y
```

### Шаг 3: Разрешение доступа к хранилищу

```bash
termux-setup-storage
```
> ⚠️ Нажми «Разрешить», когда появится запрос на доступ к файлам.

### Шаг 4: Клонирование проекта

```bash
# Перейди в домашнюю директорию
cd ~

# Клонируй репозиторий (замени URL на свой)
git clone https://github.com/your-username/anime-edit-bot.git

# Перейди в папку проекта
cd anime-edit-bot
```

Или, если проект уже на устройстве:

```bash
cd ~/anime-edit-bot
```

### Шаг 5: Создание виртуального окружения (рекомендуется)

```bash
python -m venv venv
source venv/bin/activate
```

### Шаг 6: Установка Python-зависимостей

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### Шаг 7: Настройка переменных окружения

Создай файл `.env` в корне проекта:

```bash
nano .env
```

Вставь следующее содержимое:

```env
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_telegram_id
MAX_FILE_SIZE=50
OUTPUT_FORMAT=mp4
```

Сохрани: `Ctrl+O` → `Enter` → `Ctrl+X`

## 🔑 Как получить токен бота

### Шаг 1: Найди BotFather

1. Открой Telegram
2. В поиске набери `@BotFather`
3. Нажми **Start** (Начать)

### Шаг 2: Создай нового бота

1. Отправь команду `/newbot`
2. Введи **имя** бота (например: `Anime Edit Bot`)
3. Введи **username** бота (например: `my_anime_edit_bot`)
   - Username должен оканчиваться на `bot`
   - Может содержать только латинские буквы, цифры и `_`

### Шаг 3: Скопируй токен

BotFather отправит сообщение с токеном вида:

```
1234567890:ABCDefGhIJKlmNoPQRsTUVwxyz
```

> ⚠️ **ВАЖНО:** Никому не показывай свой токен! Он даёт полный доступ к управлению ботом.

### Шаг 4: Узнай свой Telegram ID

1. Найди в Telegram бота `@userinfobot`
2. Нажми **Start**
3. Бот пришлёт твой числовой ID (например: `123456789`)
4. Впиши его в `.env` как `ADMIN_ID`

## 🚀 Запуск бота

### Обычный запуск

```bash
cd ~/anime-edit-bot
source venv/bin/activate   # если используешь venv
python -m bot.main
```

### Запуск в фоне (чтобы бот работал после закрытия Termux)

```bash
nohup python -m bot.main > bot.log 2>&1 &
```

Проверить, что бот запущен:

```bash
ps aux | grep bot.main
```

Посмотреть логи:

```bash
tail -f bot.log
```

Остановить бота:

```bash
# Найди PID процесса
ps aux | grep bot.main

# Останови процесс (замени 12345 на реальный PID)
kill 12345
```

### Запуск через tmux (рекомендуется)

```bash
# Установи tmux
pkg install tmux -y

# Создай новую сессию
tmux new -s editbot

# Запусти бота
python -m bot.main

# Отключись от сессии: Ctrl+B, затем D
# Вернись в сессию:
tmux attach -t editbot
```

## 🔧 Решение проблем (Troubleshooting)

### ❌ `ModuleNotFoundError: No module named 'aiogram'`

**Причина:** Зависимости не установлены или venv не активирован.

```bash
source venv/bin/activate
pip install -r requirements.txt
```

---

### ❌ `ffmpeg: command not found`

**Причина:** FFmpeg не установлен в Termux.

```bash
pkg install ffmpeg -y
# Проверь установку:
ffmpeg -version
```

---

### ❌ `Error: Bot token is not provided`

**Причина:** Не настроен файл `.env` или токен неверный.

```bash
# Проверь, что файл .env существует и содержит токен:
cat .env
# Должно быть:
# BOT_TOKEN=1234567890:ABCDefGhIJKlmNoPQRsTUVwxyz
```

---

### ❌ `PIL/Pillow: decoder jpeg not available`

**Причина:** Не установлены системные библиотеки для JPEG.

```bash
pkg install libjpeg-turbo -y
pip uninstall Pillow -y
pip install Pillow --no-cache-dir
```

---

### ❌ `aubio: error while loading shared libraries`

**Причина:** Не установлены зависимости для аудио-обработки.

```bash
pkg install libsndfile libfftw -y
pip install aubio --no-cache-dir
```

---

### ❌ Бот не отвечает на сообщения

**Проверь:**

1. Бот запущен? `ps aux | grep bot.main`
2. Токен верный? Попробуй создать нового бота через `@BotFather`
3. Есть интернет? `ping -c 3 api.telegram.org`
4. Посмотри логи: `tail -50 bot.log`

---

### ❌ `MemoryError` или бот вылетает при рендеринге

**Причина:** Недостаточно RAM для обработки видео.

**Решения:**
- Используй клипы покороче (до 10 секунд каждый)
- Уменьши разрешение: в `/settings` выбери 720p вместо 1080p
- Закрой другие приложения на телефоне
- Обработай меньше клипов за раз

---

### ❌ `Permission denied` при сохранении файлов

```bash
# Дай доступ к хранилищу
termux-setup-storage

# Проверь права на папки
ls -la ~/anime-edit-bot/media/
chmod -R 755 ~/anime-edit-bot/media/
```

---

### ❌ Рендеринг слишком медленный

**Советы по оптимизации:**

```bash
# Используй аппаратное ускорение (если поддерживается)
ffmpeg -hwaccels

# В config.py можно уменьшить разрешение:
# OUTPUT_WIDTH = 720
# OUTPUT_HEIGHT = 1280
```
